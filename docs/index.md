# Kaysan Bot

Kaysan AI Telegram Bot with 50+ features, 3 languages, and multi-model AI.

## Features

- Multi-language support (Kurdish, Persian, English)
- AI-powered chat with streaming
- Image generation (17+ APIs)
- Voice to text (Groq Whisper)
- Web search with smart caching
- 20+ utility tools
- Group management
- And much more...

## Quick Start

```bash
git clone https://github.com/ashkansuri-13/kaysan-bot.git
cd kaysan-bot
pip install -r requirements.txt
cp .env.example .env
python kaysan-bot/run.py
```
