## 🚀 دەستپێکردنی خێرا

### پێویستیەکان
- Python 3.12+
- ffmpeg (بۆ پرۆسەسکردنی دەنگ)
- SQLite3

### دامەزراندن

```bash
# کۆپی کردنی ڕێپۆزیتۆری
git clone https://github.com/ashkansuri-13/kaysan-bot.git
cd kaysan-bot

# دروستکردنی چواردەی هەوڵی
python3 -m venv venv
source venv/bin/activate

# دامەزراندنی پێویستیەکان
pip install -r requirements.txt

# رێکخستنی چواردە
cp .env.example .env
# فایل .env بە تووکنەکانت دەستکاری بکە

# کارپێکردنی بۆت
python kaysan-bot/run.py
```

### Docker

```bash
# دروستکردنی وێنە
docker build -t kaysan-bot .

# کارپێکردنی کۆنتێنەر
docker run -d \
  --name kaysan-bot \
  -p 8080:8080 \
  -p 9090:9090 \
  -v ./.env:/app/.env \
  kaysan-bot
```

### Kubernetes

```bash
# دروستکردنی secrets
kubectl create secret generic kaysan-secrets \
  --from-literal=bot-token=YOUR_TOKEN \
  --from-literal=openrouter-key=YOUR_KEY

# دابەشکردن
kubectl apply -f k8s/
```

## 🔧 رێکخستن

### گۆڕاوەکانی چواردە

| گۆڕاوە | وەسف | پێویست |
|--------|------|--------|
| `BOT_TOKEN` | تووکنی بۆتی تێلێگرام | بەڵێ |
| `OPENROUTER_KEY` | کلیلی API ی OpenRouter | بەڵێ |
| `OWNER_ID` | ناسنامەی بەکارهێنەری تێلێگرام | بەڵێ |
| `CHANNEL_USERNAME` | ناوی کانال | نەخێر |
| `GROQ_API_KEY` | کلیلی API ی Groq | نەخێر |
| `REDIS_URL` | ناونیشانی Redis | نەخێر |

### رێکخستنی مۆدێلەکان

| گۆڕاوە | وەسف | بنەڕەت |
|--------|------|--------|
| `PRIMARY_MODEL` | مۆدێلی سەرەکی | deepseek/deepseek-chat |
| `CHAT_MODELS` | لیستی مۆدێلەکانی چت | deepseek/deepseek-chat |
| `CODE_MODELS` | لیستی مۆدێلەکانی کۆد | deepseek/deepseek-chat |
| `VISION_MODELS` | لیستی مۆدێلەکانی بینین | openai/gpt-4o-mini |
