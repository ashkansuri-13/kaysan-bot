## 📋 معرفی

کیسان AI یک ربات هوش مصنوعی قدرتمند تلگرام است که با مدل‌های مختلف هوش مصنوعی (GPT, DeepSeek, Llama, Qwen, Gemma) از طریق OpenRouter کار می‌کند. این ربات از **۳ زبان** (کوردی سرانی، فارسی، انگلیسی) پشتیبانی می‌کند و **۸۰+ قابلیت** شامل چت هوشمند، پردازش رسانه، مدیریت گروه، ابزارهای کاربردی و سرگرمی دارد.

## ✨ قابلیت‌های کلیدی

### 🤖 چت و هوش مصنوعی
- **چت چند مدلی** — GPT, DeepSeek, Llama, Qwen, Gemma از طریق OpenRouter
- **تشخیص نیت** — انتخاب خودکار بهترین مدل برای هر درخواست
- **پشتیبانی ۳ زبان** — کوردی سرانی، فارسی، انگلیسی با تشخیص خودکار
- **حالت‌های مختلف** — معلم، برنامه‌نویس، دوست
- **حافظه مکالمه** — به یاد آوردن زمینه پیام‌ها
- **پشتیبانی کانتکست طولانی** — تحلیل اسناد و فایل‌های بزرگ
- **پاسخ‌های استریم** — خروجی بلادرنگ AI

### 🎨 رسانه و تصویر
- **تبدیل ویس به متن** — تبدیل با دقت بالا (Groq Whisper)
- **تحلیل عکس** — مدل‌های ویژن AI عکس‌ها را توصیف و تحلیل می‌کنند
- **ساخت عکس با AI** — تولید تصویر از متن
- **تبدیل متن به عکس** — تبدیل متن به تصاویر زیبا
- **ساخت میم** — تولید میم سفارشی با AI
- **تحلیل فایل** — پشتیبانی از PDF, DOCX, TXT و فایل‌های متنی

- **نظرسنجی هوشمند** — نظرسنجی با AI
- **آمار گروه** — نمایش تعداد اعضا و فعالیت
- **قوانین گروه** — تنظیم و نمایش قوانین

### 🎮 سرگرمی
- **کوییز** — مسابقه دانستنی‌ها با گزینه‌های چهارگانه
- **معما** — معماهای جذاب و سرگرم‌کننده

### 🛡️ امنیت و پایداری
- **Rate Limiting** — جلوگیری خودکار از سوءاستفاده (۳۰ درخواست/دقیقه)
- **فیلتر اسپم و کلمات نامناسب**
- **Safe Math Calculator** — محاسبه امن ریاضی بدون eval
- **Circuit Breaker** — جلوگیری از خرابی زنجیره‌ای
- **پشتیبان خودکار** از دیتابیس
- **Health Check و Readiness Probe** — پورت ۸۰۸۰
- **Prometheus Metrics** — پورت ۹۰۹۰
- **Network Monitor** — ری‌استارت خودکار در صورت خرابی

## 🚀 نصب

### روش ۱: Docker (پیشنهادی)

```bash
git clone https://github.com/ashkansuri-create/kaysan-bot.git
cd kaysan-bot
cp .env.example .env
# فایل .env رو ویرایش کنید
docker-compose up -d
```

### روش ۲: دستی

```bash
git clone https://github.com/ashkansuri-create/kaysan-bot.git
cd kaysan-bot
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# فایل .env رو ویرایش کنید
python run.py
```

## ⚙️ تنظیمات

فایل `.env` رو با اطلاعات خودتون پر کنید:

```env
BOT_TOKEN=توکن_بات_تلگرام
OPENROUTER_KEY=کلید_اپن‌روتر
OWNER_ID=آیدی_تلگرام_مالک
DEFAULT_LANG=fa
FREE_MESSAGE_LIMIT=100
DAILY_COST_LIMIT=5.0
CHANNEL_USERNAME=نام_کانال
GROQ_API_KEY=کلید_گروک
REDIS_URL=redis://localhost:6379/0
SENTRY_DSN=dsn_سنتری
```

## 🔧 متغیرهای محیطی

| متغیر | ضروری | پیش‌فرض | توضیح |
|-------|-------|---------|-------|
| `BOT_TOKEN` | ✅ | - | توکن ربات تلگرام |
| `OPENROUTER_KEY` | ✅ | - | کلید API اپن‌روتر |
| `OWNER_ID` | ✅ | - | آیدی تلگرام مالک |
| `DEFAULT_LANG` | ❌ | `ku` | زبان پیش‌فرض (ku/fa/en) |
| `FREE_MESSAGE_LIMIT` | ❌ | `100` | حد پیام رایگان |
| `DAILY_COST_LIMIT` | ❌ | `5.0` | حد هزینه روزانه ($) |
| `CHANNEL_USERNAME` | ❌ | - | نام کانال اجباری |
| `GROQ_API_KEY` | ❌ | - | کلید Groq (تبدیل ویس) |
| `REDIS_URL` | ❌ | - | آدرس Redis (کش) |
| `SENTRY_DSN` | ❌ | - | DSN سنتری (ردیابی خطا) |
